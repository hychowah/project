# Eric Chow — Portfolio

Personal portfolio showcasing robotics, embedded systems, and software projects.

Built with [Jekyll](https://jekyllrb.com/) + [Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/) theme, hosted on [GitHub Pages](https://hychowah.github.io/).

## Projects

- **Concept Tracker** — LLM-based adaptive learning system with spaced repetition and knowledge graphs
- **Autonomous Painting Robot** — Suspended robot for coating large industrial structures (C3 Construction Robotics)
- **Cable-Suspended Robot** — Patented platform for deep-foundation cleaning & inspection (US 12,398,527 B2)
- **Expandable-Wheelbase Mobile Robot** — MSc thesis: four-wheel independently-steered platform with adjustable wheelbase
- **Custom FDM 3D Printer** — Designed and built from scratch in Fusion 360

## Local Preview

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000`.
